## Ajuste recomendado con Selenium
Usa Selenium para esperar a que el contenido real aparezca y recién ahí pasar a BeautifulSoup.

### Ejemplo
```python
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from bs4 import BeautifulSoup

wait = WebDriverWait(driver, 30)

# Esperar a que desaparezca el challenge o aparezca contenido real
wait.until(lambda d: "challenge-platform" not in d.page_source.lower())

# Opcional: esperar un elemento típico de la ficha
wait.until(EC.presence_of_element_located((By.CSS_SELECTOR, "body")))

html = driver.page_source
soup = BeautifulSoup(html, "html.parser")
```

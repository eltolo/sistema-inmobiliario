## 1) Ubicación: buscar en breadcrumbs o en el bloque de datos del aviso

En Zonaprop la ubicación suele estar en un breadcrumb tipo:

- `nav[aria-label="breadcrumb"]`
- `ol.breadcrumb`
- o algún contenedor con links de zona

### Selector sugerido
```python
soup.select_one('nav[aria-label="breadcrumb"]')
```

o más flexible:

```python
soup.select_one('ol.breadcrumb')
```

### Extracción típica
```python
breadcrumb = soup.select('nav[aria-label="breadcrumb"] a, ol.breadcrumb a')
ubicacion = " > ".join([a.get_text(" ", strip=True) for a in breadcrumb])
```

Si eso no devuelve nada, buscá en el HTML por texto como:
- `Capital Federal`
- `Balvanera`
- `Combate de Los Pozos`

y revisá si está en un bloque tipo:
```html
<div class="location">
```
o
```html
<span class="section-location">
```
